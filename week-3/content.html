Welcome

Welcome to the Online Book Store demo. Create an account or login to access the Book Collection and view book details.

How to Use
  • Register (new user) or Login (existing user)
  • Forgot Password lets you reset your password after verifying email
  • After login, open Book Collection to search/filter and view details
Where Data is Stored
  • Registered users: saved in browser localStorage as wt_week3_users
  • Current login session: saved as wt_week3_current_user
Note
This is a front-end demo (no server/database). Data stays in the same browser only.
