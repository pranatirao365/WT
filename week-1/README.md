# Week 1 — HTML Frames (Online Book Store)

## How to Run
- Open `index.html` in any web browser.

## What This Week Contains
- A frames-based layout using:
  - Navigation frame (book links)
  - Content frame (book details)
  - Footer frame
- Static book detail pages (opened inside the content frame).

## Files (Quick Guide)
- `index.html` — main frameset page
- `nav.html` — navigation links
- `home.html` — welcome/home content
- `footer.html` — footer frame
- `book1.html`, `book2.html`, ... — book detail pages

## Notes
- Frames (`frameset`) are used to match the assignment requirement.
- If your browser blocks frames, open the individual HTML pages directly.
